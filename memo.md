```c
exit( FILE_OPEN_ERROR );
```


### どうしてリーダブルだと思っているかの説明

コマンドの終了ステータスは、正常終了するときは`0`、異常終了するときは
`0`以外の値を返すのが通例です。そのため、このコードは次のようにしても
通例通りの動きになります。

```c
exit(1);
```
これでも動きますが、`FILE_OPEN_ERROR`とすれば、ファイルオープンに
失敗した時にこのコードが実行されるということがわかりやすく、リーダブル
であると考えました。
